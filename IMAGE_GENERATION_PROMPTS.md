## 사용 방법

1. OpenAI 조직 인증 완료 대기 (최대 15-20분)
2. 각 프롬프트를 사용하여 이미지 생성
3. 생성된 이미지를 목표 사이즈로 리사이즈
4. 적절한 파일명으로 저장

## 리사이즈 도구

ImageMagick 또는 다른 이미지 편집 도구를 사용하여 리사이즈:
```bash
# 예시: ImageMagick 사용
convert input.jpg -resize 1200x630 output.jpg
```
